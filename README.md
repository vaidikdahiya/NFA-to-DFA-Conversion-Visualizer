# NFA-to-DFA Conversion Visualizer

An interactive web-based tool that converts Nondeterministic Finite Automata (NFA) 
to Deterministic Finite Automata (DFA) using the subset construction algorithm, 
with step-by-step visual animation.

## 📌 Features
- Define custom NFA states, alphabet, and transitions
- Automatically converts NFA to equivalent DFA
- Step-by-step animated visualization of the subset construction algorithm
- Clean and intuitive web interface
- No installation required — runs entirely in the browser

## 🛠️ Tech Stack
- **HTML** — Structure
- **CSS** — Styling
- **JavaScript** — Logic & Animation
